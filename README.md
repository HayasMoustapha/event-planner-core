# Event Planner Core API

Service cœur métier de Event Planner - Orchestration des événements et gestion des règles métier.

## Architecture

- Node.js + Express
- PostgreSQL (SQL natif)
- Intégration Auth Service externe
- Communication avec services spécialisés

## Modules

- Events
- Tickets  
- Guests
- Marketplace
- Admin

## Installation

```bash
npm install
npm start
```

## Développement

```bash
npm run dev
```

## Docker

```bash
docker-compose up -d
```
